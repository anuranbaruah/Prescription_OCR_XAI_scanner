# Frontend — Explainable Prescription Analyzer

React + Vite single-page app.

```bash
npm install
npm run dev      # http://localhost:5173 (proxies /api -> http://localhost:8000)
```

- `src/api.js` — backend client
- `src/App.jsx` — upload + results dashboard + tabs
- `src/components/` — Entities, Recommendations, Interactions, XaiGallery, BenchmarkTables

Make sure the backend is running on port 8000 (see `../backend/README.md`).
